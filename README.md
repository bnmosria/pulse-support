# Pulse — HIIT Interval Timer
 
**Support & Feedback**
 
This is the official support repository for **Pulse — HIIT Interval Timer**. If you have a bug to report, a feature to suggest, or something that doesn't look right on your device, you're in the right place.
 
## Before You Open an Issue
 
**Check if it's already reported** — browse [open issues](../../issues) to see if someone else has already reported the same thing. If they have, give it a to help us prioritise.
 
**Check the user guide** — many common questions are answered in the [User Guide](https://bnmosria.github.io/pulse/#guide).
 
## How to Get Help
 
### Found a bug?
 
Something crashed, behaved unexpectedly, or doesn't work as described? [Open a bug report →](../../issues/new?template=bug_report.yml)
 
The more detail you include — device, iOS/Android version, steps to reproduce — the faster it can be fixed.
 
### Have a feature idea?
 
Something missing that would make your training better? [Submit a feature request →](../../issues/new?template=feature_request.yml)
 
### Something looks wrong?
 
A button too small, a layout broken on your device, colours not rendering correctly? [Send UI feedback →](../../issues/new?template=ui_feedback.yml)

## What to Expect
 
- Issues are reviewed regularly
- Bug fixes are prioritised by impact and how many users are affected
- Feature requests are considered for future versions — a from others helps move them up the list
- You will receive a response on your issue when there is an update
